# Desktop Support Case Study: Recurring Windows Startup BSODs

## Overview

This repository documents the investigation and remediation of a Windows 11 desktop experiencing recurring startup Blue Screen of Death (BSOD) events, file system corruption, and Windows servicing failures.

The project follows a structured troubleshooting methodology commonly used in desktop support and IT operations environments.

## Skills Demonstrated

- Windows 11 Troubleshooting
- Event Viewer Analysis
- CHKDSK Repair
- SFC and DISM Repair Procedures
- CBS Log Analysis
- Memory Diagnostics
- SSD Health Verification
- Root Cause Analysis
- Technical Documentation
- Validation Testing
